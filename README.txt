StatQuest: Policy Lab

Files included:
- index.html
- week1.json through week7.json

How to use locally:
1. Put all files in the same folder.
2. Start a simple local server in that folder:
   python -m http.server
3. Open:
   http://localhost:8000

How to publish on GitHub Pages:
1. Create a public repository.
2. Upload all files to the root of the repository.
3. In Settings > Pages, choose Deploy from a branch, branch main, folder / (root).
4. Wait for the GitHub Pages URL to appear.
